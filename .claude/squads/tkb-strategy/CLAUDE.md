# Squad tkb-strategy — Estratégia e Visão

## Missão
Pensar além do operacional — inteligência competitiva, parcerias estratégicas, expansão de mercado, novos produtos e oportunidades que o fundador ainda não vê. O squad de estratégia funciona como um conselheiro executivo permanente.

## Agentes

| Agente | Papel | Autoridade exclusiva |
|--------|-------|----------------------|
| `@market-intelligence` | Monitora concorrentes, tendências, oportunidades de mercado | Inteligência competitiva |
| `@partnership-scout` | Identifica, qualifica e estrutura parcerias estratégicas | Desenvolvimento de parcerias |
| `@event-strategist` | Avalia eventos relevantes, prepara participação, mede retorno | Presença e networking estratégico |
| `@expansion-advisor` | Analisa novos mercados, produtos, modelos de negócio | Crescimento e diversificação |

## Contexto Estratégico

### Posição atual da TKB
- Mesa OTC focada em USDT/BRL
- Operação manual em transição para digital/API
- Mercado: Brasil, clientes PF e PJ
- Diferencial: atendimento personalizado, spreads competitivos

### Concorrentes diretos
- Outras mesas OTC brasileiras (fragmentado, maioria sem tecnologia)
- Corretoras cripto (Binance P2P, Mercado Bitcoin OTC, NovaDAX)
- Bancos digitais com câmbio (Wise, Nomad, C6 Bank)

### Oportunidades identificadas
1. **Empresas com operação internacional** (importadores, exportadores, freelas)
2. **Parceiros B2B** (corretoras, fintechs sem mesa própria)
3. **Mercado de remessas** (brasileiros no exterior, imigrantes)
4. **Tokenização de recebíveis** (futuro próximo com regulação avançando)

## Tarefas

### Radar Competitivo
```
Tarefa: competitive-radar
Agente: @market-intelligence
Frequência: semanal

Monitorar e reportar:
1. Novidades de concorrentes (produto, preço, marketing)
2. Movimentos do mercado (novos entrantes, M&A, falências)
3. Mudanças regulatórias com impacto competitivo
4. Oportunidades abertas por movimento de concorrente
5. Ameaças emergentes

Formato: briefing executivo, máx 1 página.
```

### Briefing de Parceria
```
Tarefa: partnership-brief
Agente: @partnership-scout

Para cada categoria de parceiro:
- fintechs: parceiras que podem oferecer câmbio OTC aos clientes delas
- contabilidades: acesso a empresas com dor de câmbio
- escritorios-juridicos: clientes com operações internacionais
- aceleradoras: startups com necessidade de pagamentos internacionais
- corretoras-b: corretoras de câmbio sem mesa USDT

Entregável por parceiro:
- Perfil ideal (tamanho, foco, localização)
- Proposta de valor da parceria
- Modelo de receita (comissão, white-label, referral)
- Como abordar e qualificar
```

### Calendário de Eventos
```
Tarefa: event-calendar
Agente: @event-strategist

Por trimestre, mapear:
- Eventos de cripto/blockchain no Brasil e exterior relevantes
- Conferências de câmbio, fintech e pagamentos
- Eventos de nicho (importação, exportação, offshore)
- Onde a TKB deveria estar presente (stand, palestra, networking)
- ROI estimado por evento (custo vs. oportunidade)
```

### Análise de Oportunidade
```
Tarefa: market-opportunity
Agente: @expansion-advisor

Para cada oportunidade, analisar:
1. Tamanho do mercado (TAM/SAM/SOM)
2. Fit com a TKB hoje
3. Investimento necessário
4. Prazo para receita
5. Riscos principais
6. Recomendação: perseguir agora / watch list / descartar
```

## Histórias Prioritárias

### Historia S-1: Mapa estratégico 2026
```
Como fundador TKB,
Quero ter um mapa estratégico para os próximos 12 meses,
Para tomar decisões de produto, time e investimento com clareza.

Critérios de aceitação:
- Objetivos de crescimento definidos (receita, volume, clientes)
- 3 apostas estratégicas priorizadas
- Riscos e mitigações mapeados
- Marcos trimestrais definidos
- Revisão trimestral agendada
```

### Historia S-2: Programa de parceiros B2B
```
Como TKB Asset,
Quero estruturar um programa formal de parceiros B2B,
Para escalar a distribuição sem aumentar o time de vendas na mesma proporção.

Critérios de aceitação:
- Modelo de comissão/benefício definido
- Contrato de parceria (integrado com squad legal)
- Portal do parceiro (integrado com squad tech)
- Processo de onboarding de novo parceiro
- KPIs de parceiro (volume, clientes, atividade)
```

### Historia S-3: Estratégia de entrada em remessas
```
Como TKB Asset,
Quero avaliar o mercado de remessas internacionais,
Para decidir se e como entrar nesse segmento.

Critérios de aceitação:
- Tamanho do mercado de remessas BR ↔ exterior
- Regulação específica para remessas (BCB)
- Concorrentes diretos (Wise, Remitly, Western Union)
- Vantagem competitiva da TKB nesse mercado
- Recomendação: go/no-go com justificativa
```

## Regras do Squad
- Toda recomendação estratégica deve ter dados que a suportem
- Nunca recomendar parcerias sem verificar reputação do potencial parceiro
- Análises de expansão devem sempre incluir seção de riscos
- Manter perspectiva de 3 horizontes: 0-6m, 6-18m, 18m+
- O squad estratégico informa, a decisão final é sempre do fundador
