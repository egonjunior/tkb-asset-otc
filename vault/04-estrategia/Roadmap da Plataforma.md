# Roadmap da Plataforma

---

## Fase 1 — Fundação (Concluída)
- [x] Tela de cotação OTC
- [x] Travamento de preço (lock-price)
- [x] Integração com OKX
- [x] Gestão de pedidos (TradingOrderPage)
- [x] Deploy automático via Vercel

## Fase 2 — Semi-automatizado (Atual)
- [x] Cotação em tempo real
- [ ] Upload de comprovante na plataforma
- [ ] Painel de aprovação para analistas
- [ ] Notificações por email (status do pedido)
- [ ] Histórico de operações do cliente

## Fase 3 — Automação Core (Próxima)
- [ ] Webhook PIX via parceiro bancário
- [ ] Matching automático de pagamento
- [ ] Aprovação automática (matching >= 99.5%)
- [ ] Retry automático OKX
- [ ] Alertas de risco em tempo real

## Fase 4 — API Pública
- [ ] API REST documentada para clientes B2B
- [ ] Autenticação via API Key
- [ ] Webhooks para eventos (cotação, aprovação, liquidação)
- [ ] SDK JavaScript/Python
- [ ] Dashboard de analytics para clientes

## Fase 5 — Escala LATAM
- [ ] Multi-moeda (ARS, COP, MXN)
- [ ] Multi-provedor (Binance + OKX)
- [ ] Whitelabel para parceiros
