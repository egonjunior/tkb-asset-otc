# Onboarding de Cliente

> [!abstract] Objetivo
> Ativar um novo cliente de forma segura e ágil — do primeiro contato ao primeiro trade.

---

## Etapas

### 1. Contato Comercial
- [ ] Qualificação do perfil (volume, frequência, segmento)
- [ ] Envio de proposta com spread estimado
- Responsável: [[01-squads/TKB Commercial]]

### 2. Documentação (KYC)
- [ ] Contrato OTC assinado
- [ ] Documentos PF/PJ
- [ ] Comprovante de endereço
- [ ] Declaração de origem de recursos
- Responsável: [[01-squads/TKB Legal]]

### 3. Ativação na Plataforma
- [ ] Criar conta no sistema
- [ ] Definir limite operacional
- [ ] Configurar wallets de destino
- Responsável: [[01-squads/TKB Tech]]

### 4. Primeiro Trade (acompanhado)
- [ ] Explicar interface de cotação
- [ ] Primeiro lock com suporte da equipe
- [ ] Confirmar recebimento do USDT

---

## Tempo Médio

| Etapa | Prazo |
|---|---|
| Qualificação | 1 dia |
| KYC completo | 2–5 dias |
| Ativação | 1 dia |
| **Total** | **3–7 dias** |

> [!warning] Bloqueadores comuns
> Documentação incompleta é a principal causa de atraso. Checar lista antes de iniciar.
